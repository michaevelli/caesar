## Caesar Cipher

Caesar Cipher (https://en.wikipedia.org/wiki/Caesar_cipher)  
Shift determined by first command line argument

## Usage

<pre>
$ <b>gcc caesar.c -o caesar</b>
$ <b>./caesar 2</b>
<b>shift</b>
ujkhv
<ctrl+D>
</pre>

## Installation

Compile with `gcc caesar.c -o caesar`
